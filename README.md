# JagaBersama
Jaga Bersama adalah platform full-stack aduan &amp; edukasi anti-bullying responsif untuk Gen Z Indonesia. Dibangun dengan Vanilla JS (SPA Router), Node.js + Express, dan database PostgreSQL (Supabase) serta EmailJS untuk pengiriman bukti laporan.

**Jaga Bersama** adalah platform perlindungan aman, rahasia, dan terenkripsi untuk mengadukan serta mempelajari tindakan perundungan (bullying) di kalangan Gen Z Indonesia. Project ini dibuat sebagai tugas Pendidikan Pancasila untuk membangun lingkungan digital dan sosial yang lebih ramah dan bertanggung jawab.
## 🌟 Fitur Utama
- **Single Page Application (SPA) Router**: Navigasi halaman super cepat tanpa reload menggunakan Router hash kustom di Vanilla JS.
- **Formulir Laporan 4-Langkah (Wizard)**: Pengisian data identitas (mendukung pelaporan anonim), detail kronologi, bukti tangkapan layar (upload), serta verifikasi ringkasan laporan.
- **Notifikasi EmailJS**: Pengiriman tanda terima laporan lengkap dengan nomor tracking (`JB-XXXXXXXX`) secara instan ke email pelapor.
- **Uji Pemahaman (Quiz)**: 5 pertanyaan interaktif disertai progress bar, badges skor kelulusan, dan tinjauan kunci jawaban.
- **Galeri Carousel**: Testimoni korban perundungan dan petikan motivasi dari psikolog/sosiolog (autoplay, swipe-gesture mobile, dan dots indicator).
- **Dashboard Pengguna**: Melacak semua riwayat aduan yang pernah dikirim lengkap dengan status ter-update dari admin dan menu pengaturan profil
